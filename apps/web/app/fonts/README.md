# Local Latin Font Subsets

These WOFF2 files preserve the website's existing Zhi Mang Xing display font
and Noto Serif SC body font while avoiding the unused CJK Unicode-range CSS and
assets generated by the Google-font loader for this English-language site.

- `zhi-mang-xing-latin.woff2`: Zhi Mang Xing, regular 400.
- `noto-serif-sc-latin.woff2`: Noto Serif SC, variable weight range 400-600.
- `OFL-Zhi-Mang-Xing.txt`: upstream SIL Open Font License for Zhi Mang Xing.
- `OFL-Noto-Serif-SC.txt`: upstream SIL Open Font License for Noto Serif SC.

Upstream family sources:

- https://github.com/google/fonts/tree/main/ofl/zhimangxing
- https://github.com/google/fonts/tree/main/ofl/notoserifsc

Keep the CSS variables in `../layout.tsx` and the preserved metric-adjusted
fallbacks in `../styles/font-fallbacks.css` unchanged unless a fresh byte-budget
and pixel-parity review is completed.
